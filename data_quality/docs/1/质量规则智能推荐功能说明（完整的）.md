# 质量规则智能推荐功能说明

## 1. 建设目标

质量规则智能推荐用于减少人工配置规则的工作量，尤其适用于包含大量数据库表和字段的场景。系统根据字段元数据、数据字典、样例值、业务域、分级分类和血缘关系自动判断适用规则，并生成可人工确认的推荐结果。

## 2. 实现位置

| 能力 | 文件 |
|---|---|
| 智能推荐核心引擎 | `data_quality/rules/intelligent_rule_recommender.py` |
| 智能推荐 API 门面 | `data_quality/api/intelligent_rule_recommendation_api.py` |

## 3. 输入信号

推荐引擎支持以下输入信号：

| 信号 | 用途 |
|---|---|
| 字段名称 | 识别 phone、id_card、amount、status 等语义 |
| 字段类型 | 判断字符串、数值、日期、布尔等类型 |
| 字段长度 | 推荐长度校验 |
| 样例值 | 识别手机号、邮箱、身份证、日期、IP、MAC、IMEI 等格式 |
| 元数据说明 | 辅助语义识别 |
| 数据字典 | 推荐枚举、必填、唯一、值域规则 |
| 值域 | 推荐范围校验 |
| 业务域 | 辅助字段重要性判断 |
| 分级分类 | 辅助关键字段判断 |
| 血缘关系 | 推荐关联完整性、跨系统一致性 |

## 4. 推荐能力覆盖

| 字段类型/语义 | 推荐规则 |
|---|---|
| 手机号 | 手机号格式校验、非空校验、唯一性校验 |
| 身份证号 | 身份证格式校验、非空校验、唯一性校验、出生日期一致性、性别一致性 |
| 邮箱 | 邮箱格式校验 |
| 编码 | 编码规范性校验 |
| 日期 | 日期格式校验、时间先后逻辑校验 |
| 金额 | 数值校验、非负/值域校验、精度校验 |
| 数量/工时 | 非负/值域校验、精度校验 |
| 状态/类型/等级 | 枚举值域校验、非空校验 |
| 外键/关联字段 | 关联完整性校验 |
| 跨系统血缘字段 | 跨系统一致性校验 |

## 5. 推荐结果字段

每条推荐结果包含：

| 字段 | 说明 |
|---|---|
| `recommendation_id` | 推荐结果编号 |
| `rule_id` | 推荐规则编号 |
| `display_name` | 推荐规则名称 |
| `dimension` | 评价维度 |
| `entity_type` | 适用对象 |
| `column_name` | 推荐字段 |
| `confidence` | 推荐置信度 |
| `severity` | 严重程度 |
| `validation_level` | 强弱校验级别 |
| `reason` | 推荐理由 |
| `parameters` | 推荐参数 |
| `threshold` | 推荐阈值 |
| `evidence` | 推荐证据 |
| `source_signals` | 命中的输入信号 |
| `script_preview` | SQL 脚本预览 |
| `requires_confirmation` | 是否需要人工确认 |

## 6. 推荐流程

```text
表/字段元数据
    ↓
字段语义识别
    ↓
样例值模式识别
    ↓
数据字典和值域分析
    ↓
血缘关系识别
    ↓
匹配规则库模板或生成自动规则
    ↓
计算置信度与推荐理由
    ↓
返回可人工确认的推荐结果
```

## 7. 人工确认与入库流程

推荐结果不会默认直接生效，需要人工确认。

确认时支持：

- 选择推荐规则
- 覆盖规则参数
- 调整阈值
- 设置是否启用
- 添加确认意见
- 可选自动批准

确认后写入 `RuleValidator`，形成校验记录，后续可走审批、启停和统计流程。

## 8. API 能力

智能推荐 API 位于 `IntelligentRuleRecommendationAPI`。

| 方法 | 说明 |
|---|---|
| `recommend()` | 执行智能推荐 |
| `confirm_recommendations()` | 确认推荐结果并入库 |
| `get_cached_recommendations()` | 查询缓存的推荐结果 |

## 9. 示例：执行推荐

```python
request = IntelligentRecommendationRequest(
    table=TableMetadataRequest(
        table_fqn="mysql.default.customer_order",
        table_name="customer_order",
        columns=[
            FieldMetadataRequest(
                name="mobile_phone",
                data_type="varchar",
                description="客户手机号",
                sample_values=["13800138000", "13900139000"],
            )
        ],
    ),
    min_confidence=0.6,
)

response = api.recommend(request)
```

## 10. 示例：确认入库

```python
confirm_request = RecommendationConfirmRequest(
    request_id=response.request_id,
    confirmed_by="data_admin",
    confirmations=[
        RecommendationConfirmation(
            recommendation_id=response.recommendations[0].recommendation_id,
            parameter_overrides={"mostly": "0.99"},
            threshold_override={"pass_rate": 0.99},
        )
    ],
    auto_approve=True,
)

api.confirm_recommendations(confirm_request)
```
